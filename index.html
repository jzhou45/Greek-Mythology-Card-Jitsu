OLYMPUS

CARD-JITSU

Olympus Card-Jitsu is a card game where every card has a class, a power value, and a color.

Each card either has a type of:

Monsters

Heroes

or

Gods

Class matches follow the following pattern:

Heroes slay Monsters,

Monsters terrorize Gods,

And the Gods meddle with the Heroes


In order to win a round:

You must play a card with a winning class matchup,
